GMAIL APP PASSWORD (for email to work)
================================================================================

The setup.php now pre-configures the email address (orders@omnispace3d.com)
but the App Password must be set manually after setup:

1. Log in to Google at: myaccount.google.com with orders@omnispace3d.com
2. Security → 2-Step Verification (turn on if not already on)
3. Go to: myaccount.google.com/apppasswords
4. App: Mail, Device: Windows Computer → Generate
5. Copy the 16-character password shown
6. In OmniShop Admin → Settings, find "smtp_password" and paste it in
7. Click Save Settings
8. Send a test order to confirm emails are working
