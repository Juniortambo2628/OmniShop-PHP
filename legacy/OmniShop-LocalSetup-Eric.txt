This guide walks you through setting up a local copy of the OmniShop PHP site on Susan's Windows computer.Once set up, Susan can use a single-click file to start the site any time, make changes, and preview them before anything touches the live server.What You NeedRequirementDetailsSusan's computerWindows 10 or 11OmniShop PHP filesIn the OmniShop PHP folder in Susan's Cowork folderXAMPP (free)Download from apachefriends.org — installs Apache + MySQL locallyTime neededApprox. 20–30 minutes for first setupStep 1 — Install XAMPP  (one time only)Go to: https://www.apachefriends.org/download.htmlDownload the Windows version (large orange Download button)Run the installer — accept all default optionsWhen Windows Firewall asks for permission, click Allow Access⚠  XAMPP must install to C:\xampp\  — do not change the install path or the next steps will not work.Step 2 — Copy the OmniShop Files  (one time only)Open File Explorer and go to:C:\xampp\htdocs\Create a new folder here named:  omnishopCopy the entire contents of the OmniShop PHP folder from Susan's Cowork folder into:C:\xampp\htdocs\omnishop\✓  When done correctly, the file C:\xampp\htdocs\omnishop\index.php should exist.Step 3 — Configure for Local Use  (one time only)Open this file in Notepad:C:\xampp\htdocs\omnishop\config.phpFind and update these four lines:define('DB_HOST', 'localhost');define('DB_NAME', 'omnishop_local');define('DB_USER', 'root');define('DB_PASS', '');define('SITE_URL', 'http://localhost/omnishop');⚠  Keep a copy of the live config.php (with Bluehost details) saved separately as config_live.php so it can be restored before uploading files to the live site.Step 4 — Start XAMPP and Create the Database  (one time only)Open XAMPP Control Panel (from Start Menu or C:\xampp\xampp-control.exe)Click Start next to Apache, then Start next to MySQL — both should show greenClick Admin next to MySQL — this opens phpMyAdmin in a browserIn phpMyAdmin: click New in the left sidebarDatabase name:  omnishop_local  →  click CreateStep 5 — Run Setup  (one time only)With Apache and MySQL both running, open a browser and go to:http://localhost/omnishop/setup.phpYou should see a green success screen. This creates all the database tables and the default admin login.Admin login detailsURLhttp://localhost/omnishop/adminEmailadmin@omnispace3d.comPasswordOmniAdmin2026!Every Time Susan Wants to TestDouble-click the file:  OmniShop PHP\LOCAL_TESTING\START_OMNISHOP.batThis starts Apache and MySQL automatically and opens the admin panel in the browser. No further steps needed.PageLocal URLAdmin panelhttp://localhost/omnishop/adminSolar & Storage cataloghttp://localhost/omnishop/solarandstorageGITEX Kenya cataloghttp://localhost/omnishop/gitexAfter Susan Makes Tweaks — Uploading to the Live SiteMake sure all changes look correct on the local versionRestore the live config.php (with Bluehost DB details and SITE_URL = https://shop.omnispace3d.com)Upload only the changed files to Bluehost via FTP/File ManagerTest on the live site at https://shop.omnispace3d.com⚠  Never upload the local config.php to the live site — it contains localhost database settings that will break the live site.What Susan Can TestBrowse the product catalog as a customer wouldPlace a test order and check the confirmation email arrivesLog in to the admin panel and try adding/editing productsUpload product images via Admin → ImagesCheck order management, invoices, and PDF downloads
